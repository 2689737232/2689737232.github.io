v-bind 类名 src等数据绑定的时候

图片的显示
v-bind可以简写为 “:”
javascript <div id="app"> <img v-bind:src="img"></img> </div> const vm = new Vue({ el: "#app", data: { img:"http://5b0988e595225.cdn.sohucs.com/q_70,c_zoom,w_640/images/20180314/f15d71681dbf4bcc8b33fe261d3820db.gif" } })
类名的添加 
<div id="app">
    <div :class="c">
    </div>
</div>

const vm = new Vue({
    el: "#app",
    data: {
        c:"yes"
    }
})

如果有多个类名，需要写在[]里面

<div id="app">
    <div :class="[c1,c2]">
    </div>
</div>

const vm = new Vue({
    el: "#app",
    data: {
        c1:"yes",
        c2:"no"
    }
})

通过{}进行运算

<div id="app">
     <!-- yes是添加的类名，后面的isAdd进行判断是否添加 -->
    <div :class="{yes:isAdd}">
    </div>
</div>

const vm = new Vue({
    el: "#app",
    data: {
        c:"yes",
        isAdd:true
    }
})

style的添加
如果是添加style里面的一个样式可以这样写：

<div id="app">
    <div :style="{width:500px,height:500px,backgroundColor:bgColor}">
    </div>
</div>

const vm = new Vue({
    el: "#app",
    data: {
        bgColor:"yellow"
    }
})
