VUE基础 数据绑定渲染

对对象和数组的渲染时

<div id="app">
    {{arr}}
</div>
var app = new Vue({
        el: "#app",
        data: {
            str: "str",
            arr: [1, 2, 3],
            obj: {
                a: "a",
                b: "b",
                c: "c"
            }
        }
    })

如果–arr.length，页面是不会进行刷新渲染。

如果通过数组的下标操作也不会改变。

如果使用了一个对象没有的属性值，会返回undefined。在视图中不会显示。

<div id="app">
    {{obj.yes}}    //没有这个属性值，不显示
</div>
var app = new Vue({
        el: "#app",
        data: {
            str: "str",
            arr: [1, 2, 3],
            obj: {
                a: "a",
                b: "b",
                c: "c"
            }
        }
    })

如果再添加上这个值的话，页面上是不会渲染的。

针对上面的情况可以使用 VUE的$set解决

<div id="app">
    {{arr}}
</div>
var app = new Vue({
        el: "#app",
        data: {
            str: "str",
            arr: [1, 2, 3],
            obj: {
                a: "a",
                b: "b",
                c: "c"
            }
        }
    })

$el

通过Vue实例.$el获取操作的元素。
app.$el 相当于 document.getElementById(‘app’) 

<div id="app">
    {{obj}}
</div>
var app = new Vue({
    el: "#app",
    data: {
        str: "str",
        arr: [1, 2, 3],
        obj: {
            a: "a",
            b: "b",
            c: "c"
        }
    }
})
console.log(app.$el.innerText);

更改数据，然后打印。

<div id="app">
    {{obj}}
</div>
var app = new Vue({
    el: "#app",
    data: {
        str: "str",
        arr: [1, 2, 3],
        obj: {
            a: "a",
            b: "b",
            c: "c"
        }
    }
})
console.log(app.$el.innerText); 
app.obj.a = "aaaa";
app.obj.b = "bbbb";
app.obj.c = "cccc";

页面上数据已经更改但是，打印的却不一样。

因为VUE在数据的渲染上面是，异步操作。可利用定时器等操作查看

$nextTick

这个方法是在数据渲染完成后，触发这个方法。接受一个参数

<div id="app">
    {{obj}}
</div>
var app = new Vue({
    el: "#app",
    data: {
        str: "str",
        arr: [1, 2, 3],
        obj: {
            a: "a",
            b: "b",
            c: "c"
        }
    }
})
app.obj.a = "aaaa";
app.obj.b = "bbbb";
app.obj.c = "cccc";
app.$nextTick(()=>{
    console.log(app.$el.innerText);
})
