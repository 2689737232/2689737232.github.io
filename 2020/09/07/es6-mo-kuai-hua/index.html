浏览器环境中的导入导出

需要开启liveserver，不然会有跨域问题。

需要将type类型指定为module
<script src="./xxx/xxx" type="module"></script>
导出
一个模块，默认导出只能有一个 
导出格式为 [export 声明、变量、函数或类声明。]

export let a = 45;

let xx = "xx";
export xx;

export function(){...};

export class Person(){ ... }


默认导出,只能导出一次。
let value = 666;
// 默认导出
export default value;
// 或export { Lala as default };
// 在默认导出后面可以加值。 export 45;
导入
defa为默认导出的值。可以通过as指a将定名字为b。
import defa,{xxx,a as b} from "./xxx/xxx";

如：

导出文件
export let a = { name: "a" };
let xx = "love you";

export class Person{
   constructor(name,age){
      this.name = name;
      this.age = age;
   }
}

export default xx;

导入文件
import defa[默认导出的值], { xx, xxx } from "./xx/xxx.js";
console.log(defa, xx, xxx);

注意: 在函数或其他语句中不能使用export import。
如：if(xxx){ export xx; }

如果导入的文件有很多可以使用 import * as xxx from "./xxx";全部导出。但是全部导出必须通过as重命名，因为不重命名没有名称。
如果只是想导入,只是运行一个模块可以直接使用import "./xxx"。
